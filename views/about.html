Acerca de aplicación móvil del blog TecnoLikeCuba

Esta aplicación se desarrolla haciendo uso de las nuevas generaciones de estándares para la web que se definen con el nombre de Progressive Web Apps, permitiendo de esta forma que usted pueda instalarla y usarla de manera que se comporte como nativa en su dispositivo independientemente del sistema operativo sobre el cual se está ejecutando, también permite ser agregada a la pantalla de inicio de tu dispositivo móvil o escritorio en caso de ser un ordenador.

Gracias a las tecnologías que esta utiliza para la comunicación y el intercambio de datos, permite el ahorro de estos. Almacena en tu dispositivo los archivos necesarios para poder seguir funcionando incluso en caso de que no tengas conexión a internet.
